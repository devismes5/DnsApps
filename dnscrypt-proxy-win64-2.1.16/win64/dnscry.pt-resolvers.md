# dnscry.pt Public DNSCrypt Resolvers

Last update: 2026-05-23T11:57:27Z

This file contains the DNS stamps for all public DNSCrypt resolvers operated by [dnscry.pt](https://www.dnscry.pt/). 
All servers support IPv4+IPv6, are uncensored, unfiltered, encrypted and support DNSSEC.

More information can be found at: https://www.dnscry.pt/

Get in touch: mail@dnscry.pt (PGP Key ID: `DCB22F2B`)

To use this list, add this to the `[sources]` section of your `dnscrypt-proxy.toml` configuration file:

    [sources.dnscry-pt-resolvers]
    urls = ["https://www.dnscry.pt/resolvers.md"]
    minisign_key = "RWQM31Nwkqh01x88SvrBL8djp1NH56Rb4mKLHz16K7qsXgEomnDv6ziQ"
    cache_file = "/var/cache/dnscrypt-proxy/dnscry.pt-resolvers.md"
    refresh_delay = 72
    prefix = "dnscry.pt-"

(Change `cache_file` directory if required!)
